# 개인정보 전송요구권 (§35조의2)

2025.3.13 시행. 개인정보보호법 3차 개정으로 신설.

## 권리의 내용

정보주체는 본인의 개인정보를 **다른 개인정보처리자에게 전송**할 것을 요구할 수 있다.

## 적용 대상

- 일정 규모 이상 개인정보처리자 (대통령령으로 정함)
- 표준화된 형식으로 보유한 정보
- 정형화·구조화된 데이터

## 처리방침 필수 기재

### 전송 방법

```markdown
## 개인정보 전송요구권

정보주체는 본인의 개인정보를 다른 개인정보처리자 또는 관리기관에 전송하도록 요구할 수 있습니다.

### 전송 방법
1. 마이페이지 > 개인정보 관리 > 전송 요구
2. {{contactEmail}}로 신청서 제출
3. 전화 신청: {{contactPhone}}

### 전송 가능 항목
- 회원 가입 정보
- 거래 내역 (최근 3년)
- 서비스 이용 기록

### 전송 형식
- JSON, CSV, XML 중 선택
- 표준화된 API 연동 (개보위 지정 기관)

### 전송 현황 확인
마이페이지 > 개인정보 관리 > 전송 이력에서 확인 가능
- 전송 일시
- 전송 대상
- 전송 항목
- 처리 상태

### 처리 기간
요청일로부터 **10일 이내** 처리 (근무일 기준)
```

## 전송 요구 접수 시 처리자 의무

1. **본인 확인** — 전송 요구자가 정보주체 본인인지 검증
2. **범위 확인** — 정당한 범위 내 요구인지 검증
3. **전송 실행** — 10일 이내 (법정 기한)
4. **이력 기록** — 전송 일시·대상·항목 기록 보관
5. **정보주체 통지** — 전송 완료 통지

## 거부 가능 사유

- 다른 정보주체의 권리·이익 침해 우려
- 영업비밀 침해 우려
- 본인 확인 실패

거부 시 사유를 **서면·전자우편**으로 통지.
