# 서비스 유형별 필수 조항 매트릭스

## 1. SaaS (B2B/B2C 소프트웨어 서비스)

### 필수 수집 정보
- 이메일, 이름
- 소속 회사 (B2B)
- 결제정보 (Stripe/토스페이먼츠 등)
- API 토큰 / 세션 ID

### 필수 조항
- 서비스 이용료 및 결제
- 서비스 중단 SLA
- 데이터 백업·복구
- API 사용 제한
- 소스코드·저작권 (서비스 자체)
- 이용자 데이터 소유권
- 해외 리전 데이터 저장 (AWS 등) → 국외 이전 명시

### 불필요
- 청약철회 (구독형은 해지로 처리)
- 배송 조항

### 처리위탁 예시
- AWS / Google Cloud / Vercel — 호스팅
- Stripe / 토스페이먼츠 — 결제
- SendGrid / Postmark — 이메일
- Segment / Mixpanel — 분석

## 2. 쇼핑몰 (전자상거래)

### 필수 수집 정보
- 배송지 (주소, 우편번호, 전화번호)
- 결제정보 (카드, 계좌)
- 주문·거래 내역

### 필수 조항 (전자상거래 표준약관 10023호)
- 청약철회 (7일 이내)
- 환급 절차
- 재화 공급 기한
- 구매확인·반품·교환
- 저작권
- 분쟁해결

### 관련 법령
- 전자상거래 등에서의 소비자보호에 관한 법률
- 약관규제법
- 개인정보보호법

### 보유기간 (법령 근거)
- 계약·청약철회 기록: **5년** (전자상거래법 §6)
- 대금결제 및 재화 공급 기록: **5년**
- 소비자 불만·분쟁처리 기록: **3년**
- 표시·광고 기록: **6개월**

## 3. 커뮤니티 / 소셜 서비스

### 필수 수집 정보
- 닉네임, 이메일
- 프로필 이미지 (선택)
- 게시물, 댓글

### 필수 조항
- 게시물 저작권 (이용자 귀속, 서비스 이용권 회사)
- 금지 행위 (불법, 명예훼손, 저작권 침해)
- 게시물 삭제·제재
- 신고 처리 절차
- 운영자 권한

### 14세 미만
- 서비스 성격상 청소년 이용 시 법정대리인 동의 필수
- 청소년 보호정책 별도

## 4. 블로그 / 개인 사이트

### 최소 수집 정보
- 이메일 (댓글, 문의)
- IP (로그)

### 최소 필수 조항
- 개인정보 처리방침 (필수)
- 이용약관 (선택, 광고 시 필수)
- 저작권 안내

### 법령 완화
- 소상공인·개인 운영의 경우 법 적용 일부 완화 가능
- 단, 수집·이용 시 동의·고지 의무는 동일

## 5. 핀테크 / 금융

### 추가 법령 (이중 적용)
- 신용정보법
- 전자금융거래법
- 자본시장법
- 금융소비자보호법

### 필수 조항
- 본인확인 절차 (휴대폰·공동인증서)
- 이상거래탐지 (FDS)
- 보이스피싱 보호
- 민원·분쟁해결 (금융감독원 1332)
- 금융분쟁조정위원회

### 예시
- 토스 방식의 **다중 법령 근거** 명시

## 6. AI / 자동화 서비스

### 추가 기재 (필수)
- 자동화된 결정 기준·절차·방식 (§37조의2)
- AI 학습 데이터 처리 (생성형 AI 안내서 2025.8)
- 인적 개입 재처리 요구 방법
- 학습 데이터 삭제 요구 방법

### 리스크 구분
**중대한 영향**:
- 채용 심사, 대출 심사, 보험료 자동 계산 → **거부권 필수**

**일반**:
- 추천 알고리즘, 검색 랭킹 → 설명 요구만
