# 행태정보·쿠키 처리 (2025.4 지침)

## 행태정보란

웹사이트·앱 이용 이력, 검색 이력, 구매 이력 등 **개인의 관심·성향·취향**을 알아낼 수 있는 정보.

### 수집 수단
- HTTP 쿠키
- 웹 비콘 (1x1 픽셀 추적)
- 로컬스토리지 / 세션스토리지
- 모바일 광고 식별자 (ADID Android, IDFA iOS)
- 핑거프린팅

## 처리방침 필수 기재

### 1. 수집 목적
```
- 맞춤형 광고 제공
- 서비스 이용 분석
- 부정 이용 방지
- 신규 서비스 개발
```

### 2. 수집 항목
```
- 페이지 방문 이력
- 검색어
- 클릭 이력
- 구매 이력
- 체류 시간
- 디바이스 정보 (OS, 브라우저, 화면 크기)
```

### 3. 보유·이용 기간
```
- 행태정보: 최대 1년 (맞춤형 광고 목적)
- 법령 필요 시 연장 가능
```

### 4. 제3자 제공 여부
```
- Google Analytics (미국)
- Meta Pixel (미국)
- 네이버 애널리틱스 (대한민국)
- 각 제공업체 개인정보처리방침 링크 필수
```

### 5. 거부 방법 (단계별)

**웹 브라우저**:

```markdown
### Chrome
설정 > 개인정보 및 보안 > 쿠키 및 기타 사이트 데이터
- 타사 쿠키 차단 옵션 활성화
- 전체 쿠키 차단 옵션 활성화

### Safari
환경설정 > 개인정보 보호 > 쿠키 및 웹사이트 데이터 관리
- 교차 사이트 추적 방지 활성화
- 쿠키 삭제

### Edge
설정 > 쿠키 및 사이트 권한 > 쿠키 및 사이트 데이터 관리
- 타사 쿠키 차단

### Firefox
설정 > 개인정보 및 보안 > 쿠키와 사이트 데이터
- 향상된 추적 방지 기능: 엄격 선택
```

**모바일**:

```markdown
### iOS (iPhone/iPad)
설정 > 개인정보 보호 및 보안 > 추적
- "앱이 추적 요청하지 않도록 허용" 활성화
- 설정 > 개인정보 보호 및 보안 > Apple 광고 > 맞춤형 광고 비활성화

### Android
설정 > 개인정보 보호 > 광고
- 광고 ID 삭제
- 맞춤형 광고 선택 해제
```

**시크릿/프라이빗 모드**:
- Chrome 새 시크릿 창
- Safari 새 사설 창
- Edge InPrivate 창
- Firefox 새 사생활 보호 창

## 쿠키 배너 요구사항

### GDPR 수준의 옵트인 (권장)

한국법은 필수적으로는 요구하지 않으나, 2025.4 지침은 **명시적 동의 권장**.

```
[쿠키 동의 배너]

본 사이트는 서비스 향상을 위해 쿠키를 사용합니다.

□ 필수 쿠키 (로그인, 보안) — 거부 불가
□ 분석 쿠키 (Google Analytics)
□ 광고 쿠키 (Meta, Google Ads)
□ 맞춤화 쿠키 (선호 설정)

[모두 동의] [선택 동의] [거부] [설정]
```

### 필수 쿠키 예시
- 세션 ID
- CSRF 토큰
- 장바구니 유지
- 로그인 유지

### 비필수 쿠키 (동의 필요)
- Google Analytics
- Meta Pixel
- 광고 네트워크
- A/B 테스트
