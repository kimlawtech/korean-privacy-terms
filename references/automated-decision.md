# 자동화된 결정 안내 (§37조의2)

2024.3.15 시행. AI/ML 기반 의사결정 서비스 필수 기재.

## 적용 대상

개인정보를 활용해 **정보주체의 권리 또는 의무에 중대한 영향을 미치는** 자동화된 결정을 하는 경우:
- 신용평가
- 채용 심사 (이력서 AI 평가)
- 대출 승인
- 보험료 자동 계산
- 콘텐츠 추천 알고리즘 (중대성 판단 필요)
- 부정사용자 탐지

## 처리방침 필수 기재 항목

### 1. 결정의 기준
```
예: 대출 승인 결정은 다음 데이터를 기반으로 합니다.
- 신용점수 (KCB, NICE)
- 소득 정보
- 기존 대출 이력
- 최근 6개월 연체 이력
```

### 2. 결정의 절차
```
1단계: 정보주체 데이터 입력
2단계: 자동화 알고리즘 평가 (가중치 적용)
3단계: 점수 산출
4단계: 임계값 비교 후 결과 도출
```

### 3. 결정의 방식
- 사용 AI 유형: 머신러닝/딥러닝/규칙 기반
- 학습 데이터 사용 여부
- 인간 개입 수준 (완전 자동 vs 보조 도구)

### 4. 정보주체 권리 행사 방법

**거부권**:
```
자동화 결정이 부당하다고 판단되면 [privacy@example.com]으로 거부 요청 가능.
거부 요청 시 해당 결정의 효력을 정지합니다.
```

**설명 요구권**:
```
결정 사유에 대한 설명을 요구할 수 있습니다.
요청 시 15일 이내 설명 제공.
```

**재처리 요구권**:
```
인적 검토에 의한 재처리 요구 가능.
재처리 완료 후 30일 이내 결과 통지 (최대 60일 연장 가능).
```
