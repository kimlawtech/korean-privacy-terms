# 개인정보 처리방침 작성지침 2025.4.21 개정 포인트

개인정보보호위원회가 1년 만에 개정한 최신 작성지침. 2025.6부터 처리방침 평가제에 반영.

## 6대 핵심 변경사항

### 1. 동의 vs 비동의 항목 명확화

**동의 불필요** (계약이행 목적):
- 회원 서비스, A/S 상담, 주문 배송

**동의 필수** (목적 불문):
- 민감정보 (건강, 사상, 정치적 견해 등)
- 고유식별정보 (주민번호, 여권번호, 운전면허, 외국인등록번호)
- 제3자 제공
- 마케팅·광고 이용

### 2. 항목 기재 방식 유연화

기존: 모든 항목 세부 나열
개정: **유형별 분류 허용**

예시:
```
자동화 결정 데이터:
- 자기소개서, 공인영어시험 점수, 대학 학점, 경력사항
- (유형 단위로 기재 가능)
```

보유기간도 기준만 명시 가능:
```
보유기간: 서비스 이용 종료 후 관련 법령에 따른 보유 기간 (전자상거래법 5년 등)
```

### 3. 고충처리 부서 확대

기존: 개인정보 보호책임자(CPO) 부서만
개정: **CPO 책임 하에 고객센터 등 유관부서 연락처도 기재 가능**

예시:
```
개인정보 보호책임자: 김철수 (CTO)
문의: privacy@example.com

고충처리 담당: 고객지원팀
문의: support@example.com / 1588-XXXX
```

### 4. 모바일 앱 공개 위치 다양화

기존: 홈페이지 첫 화면 하단
개정: 다음 위치 추가 허용
- 서비스 메뉴
- 설정 메뉴
- 회원가입/로그인 영역
- 앱 첫 실행 시 안내 화면

### 5. 정보주체 권리 안내 상세화

**전송요구권** (2025.3.13 시행):
- 전송 방법 (어디로, 어떻게)
- 전송 현황 조회 방법
- 전송 내역 확인 방법

**자동화 결정**:
- 결정 기준 (어떤 데이터를 어떻게 평가하는지)
- 결정 절차 (입력 → 처리 → 출력)
- 결정 방식 (알고리즘 유형, AI 학습 여부)
- 거부·재처리 요구 방법

**AI 학습 투명성** (2025.8 생성형 AI 안내서):
- 학습 데이터 출처
- 가명화·비식별화 조치
- 삭제 요구 방법

### 6. 행태정보 거부권 단계별 안내

기존: "쿠키 차단" 한 줄
개정: **단계별 상세 안내 필수**

필수 포함:
```
[행태정보 수집 거부 방법]

1. 웹브라우저 쿠키 삭제
   - Chrome: 설정 > 개인정보 및 보안 > 쿠키 및 기타 사이트 데이터
   - Safari: 환경설정 > 개인정보 보호 > 쿠키 및 웹사이트 데이터 관리
   - Edge: 설정 > 쿠키 및 사이트 권한

2. 제3자 쿠키 차단
   - 브라우저별 "제3자 쿠키 차단" 옵션 활성화

3. 모든 쿠키 차단
   - 주의: 서비스 이용에 제약 발생 가능

4. 새 시크릿 창 이용 (Chrome)
   - 이력·쿠키 저장 안 됨

5. 모바일 광고 식별자 재설정
   - iOS: 설정 > 개인정보 보호 및 보안 > 추적 > 앱이 추적 요청하지 않도록 허용
   - Android: 설정 > 개인정보 보호 > 광고 > 광고 ID 재설정
```

## 구성 권장 사항 — 라벨링 + 전문

### 라벨링 (상단 요약)
정보주체가 직관적으로 확인할 수 있도록 **아이콘 기반 6대 요약**:
1. 처리하는 개인정보 항목
2. 개인정보 처리 목적
3. 개인정보 제3자 제공
4. 개인정보 처리위탁
5. 정보주체의 권리
6. 고충처리 부서 정보

### 전문 (본문)
법 제30조 모든 필수 항목 상세 기재

## 특수 처리방침

부록으로 제공되는 특화 템플릿:
- **아동용** (14세 미만 주 이용자)
- **공공기관용**
- **소상공인용**

## 위반 시 제재

- 평가 등급 하락 (평가대상 기업)
- 작성지침 위반 지적 시 자진 시정 요구
- 반복 위반 시 과태료·과징금 가능
